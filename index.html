      An array literal is the simplest and most commonly used way to create an array in JavaScript.
       It uses square brackets [] to enclose a list of elements, separated by commas.
       Indexing starts at 0.
       Can contain elements of any data type (numbers, strings, booleans, even other arrays or objects).
       Preferred over the constructor (new Array()) for its simplicity and readability.

       Example:

       let mixedArray = ["Shrishti", 22, true, null];
       console.log(mixedArray); // Output: ["Shrishti", 22, true, null]


   
       An array constructor is another way to create an array in JavaScript using the built-in Array() function.
       let arrayName = new Array();              // Creates an empty array
       let arrayName = new Array(element1, element2, ...);  // Creates array with elements
